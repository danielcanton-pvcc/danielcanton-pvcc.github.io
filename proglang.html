Chapter 6: Programming Languages
TERM DEFINITION
Machine Language An encoding system for computers that consists of instructions of numeric digits
Debugging Process that consists of locating and correcting errors in code
Assembler A program that converts mnemonics to machine language
Assembly Language Mnemonic system for representing programs
Translator Program that translates high-level primitives to machine language
Compiler Program that compiles several machine instructions into short sequences
Interpreter Functions similar to a Translator except they execute the instructions they translate as they go
Cross-Platform Software Software that is independent of an operating system and the machine's hardware
Natural Language Common languages spoken by people, such as English, German and Spanish
Formal Language Programming Languages
Machine Independence The goal of having languages able to compile just as easily across different machines
High-level Programming Language A more powerful, abstract language that allows for more complex software designs and is easier for humans to read and write
Object-Oriented Programming (OOP) Programming Language that provides statements for describing objects
Scripting Language A programming language typically used to perform administrative tasks rather than complex programs
Data Types:
  1. Integer
  2. Float
  3. String
  1. Numeric data consisting of whole numbers
  2. Numeric data that may contain a value other than a whole number
  3. Sequence of characters, primarily uses for text
Primitive Data Type Data types that are included in primitives, such as int, char, float, and others.
Data Structure The conceptual shape or arrangement of data
Python Collection Data Structures:
  1. List
  2. Tuple
  3. Set
  4. Dictionary
  1. Used to store multiple items in a single variable
  2. Same as a list, but ordered and unchangeable
  3. Same as above, but is unordered, unchangeable and unindexed
  4. Used to store data values in key:value pairs. Ordered, changeable, and does not allow duplicates
Function A set of instructions for a task that can be used by other program units
Local Variable A variable declared within a function
Global Variable A variable that is not restricted to a particular function or part of a program
Source Program The original form of a program before it is translated
Object Program The translated form of a program
