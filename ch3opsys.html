Operating Systems Terminology
TERM DEFINITION
Operating System Software Package that coordinates a computer's internal activities
Job Queue Storage organization system that is ordered in first-in, first-out
FIFO First-in, first-out. Objects are removed in the order they arrived
Interactive Processing A program that has a dialogue with the user through remote terminals
Time-Sharing Operating systems that provide service to multiple users at the same time
Multiprogramming Time is divided in intervals, which executes jobs based on interval restrictions
Application Software Programs used to perform tasks particular to a machine
System Software Programs used to perform tasks common to computers in general
Utility Software Programs that perform activities fundamental to a computer installation
Graphical User Interface (GUI) Objects that can be manipulated are represented pictorially on a computer display
Shell An old user interface consisting of textual messages
Kernel The internal part of an operating system
Device Driver Software that communicates with controllers to utilize peripherals
File Manager Software that coordinates the machines mass storage facilities
Memory Manager Software that coordinates the machines main memory
Virtual Memory "Fictional" memory space created by paging
Boot Strapping Procedure that transfers operating system from storage into main memory
ROM Read-only memory. Special nonvolatile memory cells used to store a machines initial program
Deadlock When two or more processes are blocking each other by waiting for the same resource
Spooling A technique of holding data to output later but at a more convenient time
Super User (or Administrator) The user who can control and alter settings within an operating system
Multi-Factor Authentication A method of adding extra security by requiring more than one piece of evidence to access accounts
