Operating Systems Terminology
TERM DEFINITION
Network Classifications
  1. LAN
  2. MAN
  3. WAN
  1. Local Area Network
  2. Metropolitan Area Network
  3. Wide Area Network
Network Models
  1. Bus Network
  2. Star Network
  1. Network in which all computers are connected via a common bus
  2. Network in which all computers are connected via a hub
Protocols
  1. CSMA/CD Ethernet
  2. CSMA/CA Wifi
  1. Carrier Sense Multiple Access with Collision Detection
  2. Carrier Sense Multiple Access with Collision Avoidance
Combining Networks
  1. Repeater
  2. Bridge
  3. Switch
  4. Router
  5. Gateway
  1. Device that passes signals back and forth between two original buses
  2. Device that works like a repeater, but only sends a message destined for a computer on the opposite bus
  3. A Bridge with multiple connections, working with more than just two buses
  4. A special purpose computer for connecting networks to form an internet, and forwarding messages
  5. The spot which connects one network to an internet
Client/Server Model
  1. P2P: Peer-to-Peer
  2. Distributed System
  3. Cloud Computing
  1. Model where processes provide service to and from each other directly
  2. Software units that execute as processes on different computers
  3. Huge pools of shared computers on the network can be allocated for use by clients as needed
Internet Addressing
  1. Tier 1
  2. Tier 2
  3. Tier 3
  1. High-speed, high-capacity international WANs
  2. More regional and less potent than Tier 1
  3. Independent Internet that allows for access to the core of the internet
IP Address
  1. ICANN
  2. DNS
  3. TLD: Top-level Domain
  4. Country-code TLDs
  1. Internet Corporation for Assigned Names and Numbers
  2. Domain Name System
  3. Top-level Domains
  4. Two-letter TLD for each specific country
Internet Protocols
  1. FTP
  2. HTTP
  3. SMTP
  1. File Transfer Protocol
  2. Hyper Text Tansfer Protocol
  3. Simple Mail Transfer Protocol
World Wide Web
  1. URL
  2. Client-side Activities
  3. Server-side Activities
  1. Uniform Resource Locator
  2. Activities performed by a client
  3. Activities performed by a server
Internet Four-Layer Software
  1. Application Layer
  2. Transport Layer
  3. Network Layer
  4. Link Layer
  1. Consists of software units like clients and servers that use Internet communication
  2. Adds sequence numbers to segments it produces to allow for them to be reassembled at the message's destination
  3. Decides which direction a packet should be sent at each step of its path
  4. Deals with the particular communication details of a network in order to transfer packets
Internet is:
  1. Packet-Switching Network
  2. Fault Tolerant
  3. Redundant
  1. Transfer of small pieces of data across various networks
  2. Ability to continue operating uninterrupted despite the failure of one or more components
  3. Provides multiple pathways for traffic to keep data flowing even through a failure
Cybersecurity
  1. Malware
  2. Virus
  3. Worm
  4. Trojan Horse
  5. Spyware
  6. Phishing
  7. DDoS Attack
  1. Malicious Software
  2. Software that infects a computer by inserting itself into programs already on the machine
  3. Autonomous program that transfers itself through a network
  4. Enters a computer system disguised as another program
  5. Software that collects information about activities on the computer it resides
  6. Technique of obtaining information by asking for it
  7. The process of overloading a computer with messages
Protection
  1. Firewall
  2. Spam Filter
  3. Proxy Server
  4. Antivirus Software
  5. Cryptography
  6. Public Key Encryption
  1. Technique that filters traffic passing through a point in the network
  2. Firewall designed to block unwanted email
  3. Software unit that acts as an intermediary between client and server
  4. Software that is used to detect and remove known viruses and other malware
  5. The study of sending and receiving messages securely in the presence of adversaries
  6. Technique that encrypts and decrypts information with one known key and one secret key
